* Two-Ray Reflection Model
With Tx, height $h_t$, and Rx, height $h_r$, distance $d$ apart. We get two paths $E_{Los}$ (line of sight) and $E_i \rightarrow E_r$ being reflected off the ground.
\[E(d_1,t) = \frac{E_0d_0}{d_1}\cos(2\pi f_c(t - \frac{d_1}{c})),\ d_1 > d_0\]
 - $E_0$, $d_0$ is the reference $E$ field  at reference distance $d_0$
Los:   
 - \[d_1 = \sqrt{d^2 + (h_t - h_r)^2}\]
Reflected:
 - \[E_r(d_2, t) = \Gamma \frac{E_0d_0}{d_2}\cos(2\pi f_c(t - \frac{d_2}{c}))\]
 - \[d_2 = \sqrt{d^2 + (h_t + h_r)^2}\]
   
 - E-field at Rx is $E_{TOT} = E_{Los} + E_r$
 - Assume $d >> \lambda$
   - Also $d$ is large enough such that $\theta_i\rightarrow 0$ and $\theta_r \rightarrow 0$.
   - This means $\Gamma \rightarrow 1$ and $E_t \rightarrow 0$
 - So, $E_{TOT} = \frac{E_0d_0}{d_1}\cos(2\pi f_c(t - \frac{d_1}{c})) - \frac{E_0d_0}{d_2}\cos(2\pi f_c(t - \frac{d_2}{c}))$
 - Define path difference
   - $\Delta = d_2 - d_1 = \sqrt{d^2 + (h_t - h_r)^2} - \sqrt{d^2 + (h_t + h_r)^2}$
 - If we assume $d >> h_t + h_r$, then $\Delta \approx \frac{2h_th_r}{d}$. This gives:
   - Phase difference $\theta_{\Delta} = \frac{2\pi}{l}\Delta = \frac{2\pi f_c\Delta}{c}$
   - Time delay between E field components $\tau = \frac{\Delta}{c} = \frac{\theta_{\Delta}}{2\pi f_c}$
 - Evaluating $E_{TOT}$ at $t = \frac{d_2}{c}$, we get
   - \[E_{TOT} = \frac{E_0 d_0}{d_1}\cos(\theta_{\Delta}) - \frac{E_0d_0}{d_2} \approx \frac{E_0 d_0}{d}\left[\cos(\theta_{\Delta}) - 1\right]\]
     - $(d_1\approx d_2 \approx d)$
 - The total amplitude
   - \[|E_{TOT}(d) = 2 \frac{E_0d_0}{d}\sin(\frac{\theta_{\Delta}}{2})\]
 - For $x < 0.3 rad$, $\sin(x) \approx x$, so
   - \[\theta_D/2 = \frac{2\pi h_t h_r}{\lambda d} < 0.3\]
   - \[\Rightarrow d > \frac{20h_t h_r}{\lambda}\] (approx)
 - If this is satisfied, then:
   - \[E_{TOT}(d) \approx \frac{2E_0d_0}{d}\frac{2\pi h_t h_r}{\lambda d} = \frac{4\pi E_0 h_t h_r d_0}{\lambda d^2}\]
   - This means the power:
     - \[P_r(d) = P_t G_t G_r \frac{h_t^2h_r^2}{d^4}\]
 - There is a threshold distance:
   - \[d_f \frac{4\pi h_t h_r}{\lambda}\]
     - Fresnel distance
   - for $d < d_f,\ \frac{1}{d^2}$
   - for $d > d_f,\ \frac{1}{d^4}$

|----------+----------------------------------------------+----------------------------------------------+-----------------------------------------------|
|          | $d < h_t$                                    | $d \leq d \leq d_f$                          | $d_f < d$                                     |
|----------+----------------------------------------------+----------------------------------------------+-----------------------------------------------|
| $P_r(d)$ | $\frac{P_tK}{d^2+h_t^2}$                     | $\frac{P_tK}{d^2}$                           | $\frac{P_t K}{d^4}$                           |
|----------+----------------------------------------------+----------------------------------------------+-----------------------------------------------|
| $K$      | $G_{Los}G_{ref} \frac{\lambda^2}{(4\pi)^2L}$ | $G_{Los}G_{ref} \frac{\lambda^2}{(4\pi)^2L}$ | $G_{Los}G_{ref} \frac{h_t^2h_r^2}{(4\pi)^2L}$ |
|----------+----------------------------------------------+----------------------------------------------+-----------------------------------------------|
Note:
\[G_{Los} = \sqrt{G_{t,Los}G_{r,Los}}\]
\[G_{ref} = \sqrt{G_{t,ref}G_{r,ref}}\]
