* Linear Quadratic Regulator (LQR)
** Continuous
Define the functional
 - \[f(x,t) = \min_u\int_{t_0}^{t_f}h(x,u)\ dt = J\]
   - $t_0$ initial time
   - $t_f$ final time
Regulator:
 - $f(x, t_0) = f(x(0))$ some initial state not at equilibrium
   - $f(x, t_f) = 0$ final state is at equilibrium
\begin{cases}
\dot{x} &= g(x,u,t) \\
J &= \int_{t_0}^{t_f}h(x,u)\ dt
\end{cases}
Then by the Hamilton-Jacobi equation, J is minimised if:
\[\frac{\partial f}{\partial t} = \min_u\left[h(x,u) + \frac{\partial f}{\partial x}^Tg(x,u)\right]\]

Using [[id:129878a7-2136-473b-ac33-74da80b12e67][Linear and Time Invarient Systems (LTI)]]:
\[J = \int_{t_0}^{t_f}(x^TQx + u^TRu)\ dt\]
 - _Q_ uadratic
\[\frac{\partial f}{\partial t} = -\min_u\left[x^TQx + u^TRu + \frac{\partial f}{\partial x}^T(Ax + Bu)\right]\]

We can see this if:
Introduce: $f(x,t) = x^TPx$
 - $P$ is a square symmetric and _positive semi-definite_ matrix
    - $x^TPx\geq 0\ \forall x\neq 0$
\[\frac{\partial f}{\partial t} = \frac{\partial}{\partial t}(x^TPx) = x^T\frac{\partial P}{\partial t}x\]
\[\frac{\partial f}{\partial t}^T = (2Px)^T = 2x^TP\]
\[x^T\frac{\partial P}{\partial t}x = -\min_u\left[x^TQx + u^TRu + 2x^TP(Ax + Bu)\right]\]
To minimise $u$, we take:
\begin{align*}
\frac{\partial}{\partial u}\left(x^T\frac{\partial P}{\partial t}x\right) &= 0 \\
\Rightarrow 2u^TR + 2x^TPB &= 0 \\
\Rightarrow u_{opt} &= -R^{-1}B^TPx \\
\Rightarrow u_{opt} &= -Kx
\end{align*}
 - The optimal control law is the linear control law

Substitute $u_{opt}$ back into :TODO::
\[x^T\frac{\partial P}{\partial t}x = -x^T(Q - PBR^{-1}B^TP + 2PA)\]
For symmetric $P$, we can show:
\begin{align*}
2x^TPAx &= x^T(A^TP + PA)x \\
\Rightarrow \frac{\partial P}{\partial t} &= PA + A^TP + Q - PBR^{-1}B^TP
\end{align*}
 - Non-linear matrix differential equation
   - This is a Riccati equation
*** Riccati Equations
Find $P(t)$ by integrating backwards in time:
\[x^T(t_f)P(t_f)x(t_f) = 0\]

Kalman proved:
 - $P(t) \rightarrow P$ if $t_f -  t_0$ is large
   - This is refereed to as the infinite horizon
     - $\frac{\partial P}{\partial t} = 0$
Produces the Riccati Equation (ARE):
\[PA + A^TP + Q - PBR^{-1}BP = 0\]
 - Solve for $P$
** Discrete
\[x[n+1] = A(T_s)x[n] + B(T_s)u[n]\]
\[y[n] = Cx[n] + Du[n]\]
The discrete performance index:
\[J = \sum_{k=0}^{N-1} \left(x^T[k]Qx[k] + u^T[k]Ru[k]\right)T_s\]

At the terminal time:
\[K[N - (n+1)] = [T_sR + B(T_s)^TP[N-n]B(T_s)]^{-1}B(T_s)P[N-n]A\]
\[P[N - (n+1)] = [T_sQ + K^T[N-(n+1)]T_sRK[N-(n+1)]] + [A(T_s)-B(T_s)K[N-(n+1)]]^TP[N-n][A-BK[N-(n+1)]]\]
 - $n$ is increased from $0\rightarrow N-1$
   
In forward time, we get optimal control from:
\[U_{opt}[n] = -K[n]x[n]\]
Discrete ARE (DARE):
\[P = A^T(T_s)PA(T_s)-(A(T_s)^TPB(T_s))(R + B^T(T_s)PB(T_s))^{-1}(B^T(T_s)PA(T_s)) + Q\]
 - Infinite horizon
 - $0\rightarrow\infty$
   
Boundary Condition:
\[x^T[N]P[N]x[N]=0\]
 - Regulator

** General Input LQ Tracking - Continuous Systems
\[r \neq Au(t)\]
Augment the performance matrix:
\[J = \int_{t_0}^{t_f}[(r-x)^TQ(r-x) + u^TRu]\ dt\]
\[\dot{P} = -PA - A^TP - Q + PBR^{-1}B^TP\]
 - for infinite horizon: $\dot{P} = 0$
We also get
\[\dot{s} = (A - BR^{-1} B^TP)^Ts - Qr\]
We calculate this in reverse-time! $s(t_f) = 0$
 - $s$ is called the tracking vector
\[u_{opt} = -R^{-1}B^TPx - R^{-1}B^Ts\]
If we let $v = -R^{-1}B^Ts$ and $K = R^{-1}B^TP$, then:
\[u_{opt} = -Kx + v\]

** General Input LQ Tracking - Discrete Systems
\[J = \sum_{k=0}^{N-1}\left([r[k] - x[k]]^TQ[r[z]-x[k]] + u^T[k]Ru[k]\right)T_s\]

Discrete minimisation of $J$ gives discrete recursive Riccati Equations:
\[K[N-(n+1)] = [T_sR + B(T_s)^TP[N-n]B(T_s)]^{-1}B(T_s)^TP[N-n]A(T_s)\]
$P$ is defined as:
\[P[N-(n+1)] = [T_sQ + K^T[N-(n+1)]T_sRK[N-(n+1)]] + [A(T_s)-B(T_s)K[N-(n+1)]]^TP[N-n][A(T_s)-B(T_s)K[N-(N+1)]]\]
 - These are ran in reverse time together with the discrete reverse-time state tracking equation:
\[s[N-(n+1)] = F(T_s)s[N-n] + G(T_s)r[N-n]\]
 - $s[N] = 0$
Where $F(T_s)$ and $G(T_s)$ are the discrete transition and control matrices, in continuous time these are:
\[F = (A-BR^{-1}B^TP)^T\]
\[G = -Q\]
discrete versions are (using $F(T_s) = e^{FT_s}$):
\[F = I + FT_s + \frac{F^2T_s^2}{2!} + \frac{F^3T_s^3}{3!} + \hdots\]
\[G = \sum_{K=0}^{\infty}\left(\frac{F^k T_s^{k+1}}{(k+1)!}\right)G\]
The command vector takes the form:
\[v[N-n] = -R^{-1}B^Ts[N-n]\]

When ran in forward time, the optimal control at $n=kT_s$ is:
\[u_{opt}[n] = v[n] - K[n]x[n]\]
and we update the state through:
\[x[n+1] = A(T_s)x[n] + B(T_s)u_{opt}[n]\]
