* How to produce latex pdf with citations
 - Make a ~.bib~ file with the citations
 - Add \cite{*CITATION*} to the citation point
 - Add \newpage and \printbibliography to the end of the ~.org~ file
 - run ~xelatex *NAME*~
 - run ~biber *NAME*~
 - run ~xelatex *NAME*~ twice
