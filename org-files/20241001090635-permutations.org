* Permutations
 - The amount of times a set of objects can be ordered
   - We use the factorial
   - \[n!\]
 - If we dont want to include all of permutations
   - \[P_k^n=\frac{n!}{(n-k)!}\]
   - $k$ objects from a set of $n$ objects
