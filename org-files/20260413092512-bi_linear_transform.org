* Bi-linear Transform
\[z = e^{T_ss} \leftrightarrow s = \frac{1}{T_s}\ln(z)\]
 - Exact
\[z = \frac{as + b}{cs + d} \leftrightarrow s = \frac{-dz+b}{cz-a}}\]
 - Approximate

** Transform that preserves stability
\[s = \frac{z+1}{z-1} \Leftrightarrow z = \frac{s+1}{s-1}\]
 - Only for stability assessment

** Tustin Transform
Preserves dynamic behaviour with reasonable accuracy
\[s = \frac{2(z-1)}{T_s(z+1)} \Leftrightarrow z = \frac{1 + \frac{T_s}{2}s}{1 - \frac{T_s}{2}s}\]
