* Lowpass to Highpass Transform
Using the transform
 - \[s=\frac{\omega_{L1}\omega_{H2}}{\lambda}\]
 - This allows
   - \[|H(\omega_{H2})|=|G(\omega_{L1})|\]
   - \[|H(\omega_{H1})|=|G(\omega_{L2)|}\]
We can transform lowpass filters to highpass filters
