* The Q Function
\[Q(x) = \int_x^{\infty}\frac{1}{\sqrt{2\pi}e^{-\frac{\lambda^2}{2}} d\lambda\]
 - Can use Q tables to solve
   
** Alternative Form (Craig's Form)
\[Q(x) = \frac{1}{\pi} \int_0^{\frac{\pi}{2}}e^{-\frac{x^2}{2\sin(\theta)}}d\theta\]
 - $x \geq 0$
