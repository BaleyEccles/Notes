* Modulation Index
** In [[id:c9b76a54-da68-4891-9ed1-3d64a182d026][Amplitude Modulation]]
 - Defined as
   - \[a = |\min(m(t))|\]
   - \[a = \frac{E_{max}- E_{min}}{E_{max}+E_{min}}\]
 - The modulation is linear if $a \leq 1$
   - Otherwise it is non linear
