* Link Budgeting
The universal link budget:
\[P_r[dBw] = P_t[dBw] + \sum dB\ Gains - \sum dB\ Losses\]
For SNR:
\[\frac{S}{N} = P_r[dBw] - P_N[dBw] = P_t[dBw] + \sum dB\ Gains - \sum dB\ Losses - P_N[dBw]\]
Receiver sensitivity = $P_N[dB] + \frac{S}{N}[dB]$
