* Multiplexers

** Select Line
Binary number
 - $S_0$, $S_1$ ...
 - $2^n$ possible inputs
** Data Inputs
The input data that will be passed through depending when the Select Line is correct

** Table
| Data Select Inputs |     | Input Selected |
|                S_1 | S_0 |                |
|                  0 |   0 | D_0            |
|                  0 |   1 | D_1            |
|                  1 |   0 | D_2            |
|                  1 |   1 | D_3            |
